# LocalCanvas 快速入门

> LocalCanvas 使用远程 HTTP API（火山方舟 Seedream/Seedance、OpenAI 兼容、Replicate 等）进行 AI 生成，无需安装 ComfyUI 或其他本地推理环境。

## 1. 安装与启动

### 从安装包（推荐）

1. 下载并运行 `LocalCanvas-Setup-x.x.x.exe`（Windows）或挂载 `.dmg`（macOS）
2. 首次启动会进入引导向导

### 从源码开发

```bash
git clone https://github.com/localcanvas/localcanvas
cd localcanvas
npm install
npm run dev
```

## 2. 首次配置

1. 在引导界面或 **设置 → 图像/视频** 中配置 API Key
2. 推荐先添加火山方舟 Seedream（图像）与 Seedance（视频）预设
3. 点击「测试连接」确认可用
4. 在 **设置 → 通用** 中配置输出目录与 FFmpeg（视频合成需要）

## 3. 第一次创作

1. 在开始页 **新建项目**
2. 双击画布空白处 → 创建 **文本节点**，输入画面描述
3. 从文本节点拖线到 **图片节点** → 选择模型 → 点击「生成」
4. 将图片节点连线到 **视频节点** → 生成短视频
5. 项目每 30 秒自动保存，失焦时也会保存

## 4. 工作流模板

1. 打开侧边栏 **工具** 标签
2. 选择预置模板（如「文生图 → 图生视频」）点击 **加载**
3. 选中多个节点 → 右键 **打组** → **保存为工作流** 可保存自定义模板

## 5. 生成历史

1. 侧边栏 **历史** 标签可查看所有生成记录
2. 支持按类型筛选、搜索提示词
3. 点击 **复用** 可在画布中心创建带相同参数的新节点

## 6. 视频合成

1. 添加多个 **视频节点** 并生成片段
2. 添加 **合成节点**，将视频/音频连线到对应端口
3. 在合成面板配置顺序后导出 MP4

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 图像生成失败 | 检查 API Key、模型是否已开通 |
| 视频合成不可用 | 设置中检测或下载 FFmpeg |
| 磁盘空间不足 | 清理输出目录或更换输出路径 |
| 关闭时提示未保存 | 选择「保存并退出」或「不保存」 |

更多模型配置细节见 [模型配置指南](./model-config.md)。
