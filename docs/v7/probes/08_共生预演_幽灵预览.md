## 3. 状态机

```
Scheduler: idle --signal+debounce 3s--> armed --budget ok--> running
running --success--> shown --promote--> promoted
shown --dismiss/30s--> dismissed
armed --breath!=hold--> cancel (若 REQUIRE_HOLD=true)
```

**Main 进程单例**：每项目同时仅 1 个 probe task。

## 4. 数据模型

```typescript
interface GhostPreview {
  id: string
  projectId: string
  thumbPath: string
  assetPath: string
  compiledPrompt: string
  resonanceHash: string
  directionIndex: number
  status: 'pending' | 'shown' | 'promoted' | 'dismissed'
  createdAt: string
}

interface ProbeBudget {
  projectId: string
  date: string // YYYY-MM-DD
  used: number
  limit: number
}
```

```sql
CREATE TABLE ghost_previews (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  thumb_path TEXT NOT NULL,
  asset_path TEXT NOT NULL,
  compiled_prompt TEXT NOT NULL,
  resonance_hash TEXT NOT NULL,
  status TEXT NOT NULL,
  created_at TEXT NOT NULL
);

CREATE TABLE probe_budget (
  project_id TEXT NOT NULL,
  date TEXT NOT NULL,
  used INTEGER NOT NULL DEFAULT 0,
  limit_count INTEGER NOT NULL DEFAULT 20,
  PRIMARY KEY (project_id, date)
);
```

### 低分生成参数

```typescript
const PROBE_IMAGE_PARAMS = {
  width: 512,
```

### resonanceHash

```typescript
function hashResonance(field: ResonanceField): string {
  return sha256(JSON.stringify(field.sources.map(s => [s.id, s.gravity]))).slice(0, 16)
}
```

相同 hash 5min 内不重复 probe。
