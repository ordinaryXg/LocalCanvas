## 1. 用户旅程

### 局部负熵

| 步 | 用户 | 系统 |
|----|------|------|
| 1 | 预览区聚焦，点「去掉假的」 | `negentropy:detect` |
| 2 | 见 3–6 条 checklist | Vision LLM 8s 内 |
| 3 | 勾选 2 条 | 预览 diff 面板 |
| 4 | 确认 | 从 prompt 删 token + negative 追加 |
| 5 | 自动触发追加生成 | superposed append |

### 雕刻模式（Phase 1）

| 步 | 系统 |
|----|------|
| 1 | `steps=8` 混沌图 |
| 2 | 用户画矩形 | 存 mask PNG |
| 3 | inpaint 或整图重生成 | 新候选 |

## 3. 状态机

```
idle --detect--> detecting --ok--> review --apply--> generating --> idle
detecting --fail--> idle (toast)
review --cancel--> idle
```

**并发**：detecting 时禁用生成钮。

## 4. 数据模型

```typescript
interface FakeElement {
  id: string
  label: string
  reason: string
  promptTokensToRemove: string[]
  negativeTerms: string[]
  confidence: number
}

interface NegentropyDetectResult {
  items: FakeElement[]
  modelId: string
  createdAt: string
}

interface NegentropySession {
  id: string
  shotSlotId: string
  baseAssetPath: string
  iterations: Array<{
    removedIds: string[]
    promptBefore: string
    promptAfter: string
    resultAssetPath?: string
  }>
  status: 'carving' | 'done'
}
```

会话表可选；Phase 0 仅内存 + history。

### detect Prompt（Vision）

```
分析图像中显得虚假或不自然的可见元素。JSON:
{ items: [{ id, label, reason, promptTokensToRemove[], negativeTerms[], confidence }] }
不要建议改变整体风格，只列可减法项。至少 3 项，最多 8 项。
```

## 8. 风险

| 风险 | 对策 |
|------|------|
| Vision 幻觉假项 | 显示 confidence，默认不勾选 <0.5 |
| 删词过度 | 预览 diff 必显 |

**降级**：无 vision 时仅对 prompt 文本 LLM 分析。
