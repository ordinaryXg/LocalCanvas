# 探针 07 — 负熵创作：减法雕刻

> **概念锚点**：从混沌删减，而非堆叠；删除与生成同权。  
> **实验 E4**：「删除假的」按钮点击率 > 同屏「再生成」的 50%。

---

## 一、交互稿

### 1.1 两种模式

| 模式 | 入口 | 流程 |
|------|------|------|
| **雕刻模式** | 节点工具栏「🪨 雕刻」 | 从噪声/粗稿出发 |
| **局部负熵** | 预览区「去掉假的」 | 对现有候选做减法 |

### 1.2 雕刻模式流程

```
1. 生成混沌底稿（低步数 img2img / 随机分镜句群）
2. 用户圈选区域 或 点选「假」清单项
3. AI 返回 mask + 修订 prompt（subtract 语义）
4. inpaint / 文本删减迭代
5. 重复直到用户点「够硬了」
```

### 1.3 「假」清单 UI

Vision LLM 先列 **可删元素**（非改图）：

```
□ 塑料感皮肤高光
□ 多余手指
□ 背景广告牌文字
□ 过度 HDR 光晕

[ 删除选中 ]  [ 全部删掉试试 ]  [ 取消 ]
```

勾选 → 生成 `negativePrompt` + 局部 mask 建议。

### 1.4 与叠加态

- 雕刻产生的新候选 **替换** 被选中的鬼影，或作为新鬼影追加
- 删除假元素成功的层 → 自动记入隐迹（负向，`eventType: reject`）

## 三、技术探针

### 3.1 MVP（3 天）

**阶段 1：仅文本负熵**（图片节点 prompt）

1. 按钮「去掉假的」→ LLM 列出 5 条 `fakeElements`
2. 用户勾选 → `negativePrompt += ...` + 从正向 prompt 删对应词
3. 触发普通 regenerate（探针 03 追加候选）

**阶段 2：图像 mask**（可选）

- 若有 inpaint API：用 SAM 式矩形 mask（手动画框）
- 无则跳过，仅文本负熵也足以验证 E4

### 3.2 LLM Prompt

```typescript
const FAKE_DETECT = `列出画面中"显得假"的可见元素，JSON: { items: [{ id, label, reason, promptTokensToRemove[] }] }`
```
