# 探针 02 — 共鸣场：调谐盘

> **概念锚点**：用共振源调谐跨模态风格向量，取代堆 prompt。  
> **实验 E1**：用户 7 日内调谐盘交互次数 > 打开纯 prompt 框次数。

---

## 一、交互稿

### 1.1 共振源类型

| 类型 | 输入方式 | 提取结果 |
|------|----------|----------|
| `hum` | 麦克风录制 ≤15s | 节奏密度、基频轮廓 |
| `blur_image` | 拖入图片 | 色温、对比度、模糊核 |
| `phrase` | 单行文本 ≤80 字 | 隐喻核、情感效价 |
| `clip_3s` | 从视频/时间线裁 3s | 运动向量、剪辑节奏 |

### 1.2 调谐盘 UI

- **中心**：项目当前合成向量（不可直接编辑，只读光球）
- **轨道**：每颗共振源 = 一颗行星
  - **距离中心** = 权重 `gravity` 0–1（近强远弱）
  - **轨道速度** = 活性（最近是否被调整）
  - **颜色** = 提取的主色温
- **手势**：
  - 拖行星半径 → 改 `gravity`
  - 双击行星 → 预览提取摘要 + 删除
  - 拖入新素材到盘外缘 → 创建新行星，自动跑提取

### 1.3 注入生成

所有 `image` / `video` / `text` 生成请求经 `resonanceCompiler.compile()`：

```
最终 prompt = baseIntent
           + styleVectorToPrompt(vector)
           + negativeFromRemovedFrequencies()
```

用户**看不见**完整 prompt，除非「下潜」引擎视图。

### 1.4 线框

```
        ╭─── phrase:「雨夜心悸」───╮
        │         ○               │
        │    ○         ● 核心光球  │
        │         ○               │
        ╰─── hum:8s ──────────────╯
              ↑ 拖拽调整引力

  [ + 添加共振源 ]     引力总和自动归一化
```

## 二、数据模型

```typescript
interface ResonanceSource {
  id: string
  projectId: string
  type: 'hum' | 'blur_image' | 'phrase' | 'clip_3s'
  assetPath?: string          // 媒体路径
  text?: string
  /** 提取向量 64 维 float，JSON 存储 */
  vector: number[]
  /** 可解释摘要，展示给用户 */
  summary: {
    colorTemp?: number
    arousal?: number
    valence?: number
    metaphor?: string
    rhythmBpm?: number
  }
  gravity: number             // 0-1
  createdAt: string
}

interface ResonanceField {
  projectId: string
  sources: ResonanceSource[]
  /** 合成向量 = Σ gravity_i * vector_i / Σ gravity */
  blendedVector: number[]
}
```

**SQLite**

```sql
CREATE TABLE resonance_sources (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  type TEXT NOT NULL,
  payload TEXT NOT NULL,
  vector TEXT NOT NULL,
  summary TEXT NOT NULL,
  gravity REAL NOT NULL DEFAULT 0.5,
  created_at TEXT NOT NULL
);
CREATE INDEX idx_resonance_project ON resonance_sources(project_id);
```
