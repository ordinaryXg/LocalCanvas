### 4.2 compilePrompt 规则（可单测）

```typescript
export function compilePrompt(base: string, field: ResonanceField): CompiledPrompt {
  const s = field.sources
  const blended = blendSummaries(s)
  const parts = [
    base,
    blended.metaphor && `mood: ${blended.metaphor}`,
    blended.colorTemp < 4500 && 'cool color grading, low key',
    blended.colorTemp > 6500 && 'warm golden light',
    blended.arousal > 0.7 && 'high energy, dynamic',
    blended.valence < -0.2 && 'melancholic atmosphere',
  ].filter(Boolean)
  const negative = s
    .filter(x => x.gravity < 0.15)
    .map(x => `avoid ${x.summary.metaphor}`)
    .join(', ')
  return { prompt: parts.join(', '), negativePrompt: negative }
}
```

## 8. 风险与降级

| 风险 | 对策 |
|------|------|
| LLM 提取慢 | phrase 先同步显示行星，summary 异步填充 |
| 编译 prompt 过长 | 限 500 字符，超出截断 + warn |
| 无 LLM | phrase 原文直接作 metaphor，禁用 image extract |

**降级**：`VITE_FLUID_RESONANCE=0`，生成器行为与 v6 一致。
