# 探针 11 — 创作相位：界面变形

> **概念锚点**：界面随探索/收敛/冻结相变形；技术视图是地下室。  
> **假设**：默认 UI 不出现节点，老用户仍可通过「下潜」访问引擎。

---

## 一、交互稿

### 1.1 三相定义

| 相位 | 自动条件 | 主 UI | 隐藏 |
|------|----------|-------|------|
| **探索 explore** | 无 primary 镜头 / 温度>0.6 | 雾池 + 调谐盘 + 地形 | 时间轴、端口 |
| **收敛 converge** | ≥1 叠加态 / 0.3<温度≤0.6 | 鬼影胶片 + 合唱团 | 模型名 |
| **冻结 freeze** | 结晶比>0.5 或手动 | 时间轴 + 导出 | 调谐盘（可收起） |

手动切换：右上角 `○探索 ●收敛 ○冻结`（不阻止自动）。

### 1.2 布局变形

**探索相**

```
┌────────调谐盘(左)────────┬────────雾池/地形(主)────────┐
│                          │                            │
│  共鸣轨道                │  情感地形 2.5D              │
└──────────────────────────┴────────────────────────────┘
│ 呼吸覆层 │ 幽灵卡片                                      │
```

**收敛相**

```
┌────────合唱团(左)────────┬────────叠加态预览(主)────────┐
│  混音台 + 台词           │  扇形展开 / 负熵工具           │
└──────────────────────────┴────────────────────────────┘
```

**冻结相**

```
┌──────────────── 合成剪辑台（复用 ComposeEditor）─────────┐
│  时间轴 + 结晶标记 + 导出                                 │
└──────────────────────────────────────────────────────────┘
```

### 1.3 下潜引擎

- 快捷键 `` ` `` 或设置「显示引擎地下室」
- 平滑过渡：液态 UI 淡出 200ms → 现有 `Canvas.tsx` React Flow
- 面包屑：`客厅 / 地下室` 一键返回

### 1.4 首次引导

15s 可跳过动画：热=探索、凉=冻结、导出=拍照。

## 二、数据模型

```typescript
interface UiPhaseState {
  phase: 'explore' | 'converge' | 'freeze'
  phaseSource: 'auto' | 'manual'
  basementVisible: boolean
```

## 三、技术探针

### 3.1 MVP（2 天）

1. `FluidShell.tsx` 路由包装 `App` 主内容区
2. `useUiPhase()` 读 `fluidStore` 计算相位
