## 3. 状态机

```typescript
function derivePhase(fluid: FluidState, stats: FluidStats, manual?: UiPhase): Phase {
  if (manual?.locked) return manual.phase
  if (stats.crystallizedRatio > 0.5 || fluid.phase === 'freeze') return 'freeze'
  if (stats.primaryCount === 0 || fluid.temperature > 0.6) return 'explore'
  if (stats.superposedCount > 0) return 'converge'
  return fluid.phase
}
```

`basementVisible` 独立布尔，不随 phase 自动关闭。

## 4. 数据模型

```typescript
interface UiPhaseState {
  phase: 'explore' | 'converge' | 'freeze'
  phaseSource: 'auto' | 'manual'
  phaseLocked: boolean
  basementVisible: boolean
  sidebarCollapsed: { left: boolean; right: boolean }
}
```

持久化 `localStorage` per project：`ui_phase_{projectId}` 仅 manual 偏好。
