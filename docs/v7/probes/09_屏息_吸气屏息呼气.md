> **实验 E5**：与探针 08 共用满意度指标。

## 0. 落地原型边界

### Phase 0（2 天）

- `breathStore` + 覆层动画 + 3s 定时器
- Agent 全屏 plan 门控

### Phase 1（+1 天）

---

## 1. 用户旅程

| 步 | 用户 | 系统 |
|----|------|------|
| 1 | 输入 phrase 并 blur | phase=inhale，边缘脉动 |
| 2 | 停 3s | phase=hold，中央细线 |
| 3 | probe 完成 | phase=exhale，幽灵出现 |
| 4 | 点击幽灵或 30s 无操作 | phase=idle |
| 5 | 再输入 | 重置 inhale |

**主动屏息**：按 `H` → 直接 hold（跳过 inhale 时长）。

## 3. 状态机（完整）

```typescript
type BreathPhase = 'idle' | 'inhale' | 'hold' | 'exhale'

// 转移表
const transitions: Record<BreathPhase, Partial<Record<string, BreathPhase>>> = {
  idle: { input: 'inhale', keyH: 'hold' },
  inhale: { input: 'inhale', timeout3s: 'hold', keyH: 'hold' },
  hold: { input: 'inhale', probeReady: 'exhale', keyH: 'hold' },
  exhale: { confirm: 'idle', timeout30s: 'idle', input: 'inhale' },
}
```

**input 定义**：resonance patch、phrase blur、affect mouseup、chorus apply。

## 4. 数据模型

```typescript
interface BreathState {
  phase: BreathPhase
  inhaleStartedAt: number | null
  holdStartedAt: number | null
  exhaleStartedAt: number | null
  inhaleEventIds: string[]
}
```

Zustand only；telemetry 可选写 `fluid_events`：`breath_cycle_complete` 含各相时长。
