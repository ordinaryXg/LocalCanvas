# 探针 05 — 隐迹记忆：考古层

> **概念锚点**：记忆是多层字迹；被否方案以隐喻残留影响后来。  
> **实验 E6**：用户主动「召回否决议题」后，7 日内再次召回率 >40%（说明有惊喜感）。

---

## 一、交互稿

### 1.1 考古层可视化

侧栏 **「地层」** 视图：垂直剖面，每层 = 一次创作事件。

```
  ▲ 新
  │ ┌─ 层7：拒绝眼神特写（隐喻：孤独）────────┐
  │ ┌─ 层5：红夹克设定（已废弃，残留：红色）──┐
  │ ┌─ 层2：初稿 brief「雨夜」────────────────┐
  ▼ 旧
```

- 层可 **静音**（不参与召回）或 **复活**（提升权重）
- 双击层 → 展开：原图缩略、拒绝原因、AI 提取的 `metaphorTags`

### 1.2 输入：拒绝即归档

当用户：
- 在叠加态扇形按 `X` 删除候选
- 点历史记录「不要这个」
- 说「这太假了」

→ 弹出轻量条：`记入地层？[ 原因可选 ]` 默认 3s 自动记入。

### 1.3 召回交互

**自然语言**：「让怀旧者看看这个」 / 「用第 7 次那种忧郁」

解析为：
```typescript
recallQuery({ layerHint: 7, voice: 'nostalgic', tags: ['melancholy'] })
```

返回 Top-3 层 → 预览「若复活，prompt 将添加…」→ 用户确认。

### 1.4 与生成联动

复活层注入 **非事实性** 修饰（避免脸崩）：

```
Apply metaphor only: "isolation, distant gaze" — do NOT replicate exact costume from layer 5
```

## 二、数据模型

```typescript
interface PalimpsestLayer {
  id: string
  projectId: string
  depth: number              // 自动递增
  eventType: 'reject' | 'abandon' | 'brief' | 'manual_note'
  assetPath?: string
  textSnapshot?: string
  userReason?: string
  /** AI 提取，可检索 */
  metaphorTags: string[]
  emotionalSignature: { arousal: number; valence: number }
  /** 0=静音 1=完全参与召回 */
  vitality: number
  createdAt: string
}
```

**SQLite**

```sql
CREATE TABLE palimpsest_layers (
  id TEXT PRIMARY KEY,
  project_id TEXT NOT NULL,
  depth INTEGER NOT NULL,
  event_type TEXT NOT NULL,
  payload TEXT NOT NULL,
  metaphor_tags TEXT NOT NULL DEFAULT '[]',
  emotional_signature TEXT NOT NULL,
  vitality REAL NOT NULL DEFAULT 1,
  created_at TEXT NOT NULL
);
CREATE INDEX idx_palimpsest_project_depth ON palimpsest_layers(project_id, depth DESC);
```

## 三、技术探针

### 3.1 MVP（3 天）

1. **拒绝钩子**：接探针 03 `archive()` + history reject 按钮
2. **LLM 提取**（单次 JSON）：

```typescript
// electron/utility/services/palimpsest/extract-layer.ts
const prompt = `从以下被拒绝的创作片段提取 metaphorTags[3-5] 和 emotionalSignature，不要描述具体人脸服饰细节...`
```

3. **召回检索**：先 `metaphorTags` 重叠度 + `emotionalSignature` 欧氏距离；无 LLM
4. `PalimpsestStrataPanel.tsx` 列表 + 剖面简图

### 3.2 IPC

```
palimpsest:appendLayer
palimpsest:listLayers
palimpsest:recall
palimpsest:setVitality
```

### 3.3 与共鸣场/合唱团

- 召回结果可 **推为一枚临时行星**（低引力），用户可调
- 合唱团「怀旧者」声部权重 >0.6 时，召回排序 `brief` 和 `abandon` 层加权

## 四、验收（E6）

| 指标 | 通过线 |
|------|--------|
| 自动记入 | 拒绝候选后地层 +1 |
| 召回 | 「忧郁」查询返回含 emotional 匹配层 |
| 惊喜 | 用户调研：召回后 3/5 说「没想到还记得」 |

**隐私**：设置 → 清空地层；导出项目可选不含 `palimpsest_layers`。
