## 核心交付

| 模块 | 要点 |
|------|------|
| 用户系统 | 注册/登录/游客、按 `user_id` 隔离、v4 遗留迁移 |
| Agent | 意图 → 计划 → 落画布 → 可选自动 DAG |
| DAG | 拓扑执行、进度面板、重试/跳过/执行到此节点（v9 补齐） |
| 分镜组 | 脚本转分镜、宫格、批量重生、导出 |
| 后期 | 人声分离、SRT 导入/烧录、风格预设 |
