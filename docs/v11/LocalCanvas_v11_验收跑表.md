## 三、名词对照（看表时不懵）


| 文档里的词          | 人话                                           |
| -------------- | -------------------------------------------- |
| Plan 模式        | 没选中画布节点时：让 AI **规划**整片或推荐模板                  |
| Build 模式       | 选中某个节点后：让 AI **在现有图上改**（加视频、加节点）             |
| WorkflowPlan   | **简单片**计划：通常 text→image→video 几条节点           |
| ProductionPlan | **Studio 制作计划**：带简报、镜头表、脚本骨架、合成位             |
| checkpoint     | 落盘计划后 **不自动跑完全部生成**，等你审一步再走                  |
| Handoff 条      | 计划落盘后 Agent 下方的 **「下一步去哪」导航**（①脚本 ②分镜…）      |
| GraphPatch     | **画布补丁**：不重建整图，只在锚点旁加节点/连线                   |
| skeleton 落盘    | 一次放上 **脚本 + 空分镜组 + 合成 + BGM 占位**，边先不连满       |
| expandShots    | 把镜头表 **展开成** 每镜 text→image→video 小链条（最多 6 镜） |
| Take / 选用版     | 同一分镜 **多个版本**，选一个当「正式版」进合成                   |
