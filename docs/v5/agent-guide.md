# LocalCanvas Agent 使用指南

Agent 是侧边栏中的 AI 工作流助手：你用自然语言描述目标，它会生成 **工作流计划**（节点 + 连线），按能力目录 **自动选模**，确认后落到画布。部分计划还支持确认后 **自动 DAG 执行**。

## 快速开始

### 前置条件

1. 在 **设置 → 🤖 Agent** 查看就绪状态，并在 **已接入模型 / 默认模型** 中至少配置：
   - 一个 **LLM**（默认 LLM，用于文本节点与 LLM 规划）
   - 推荐同时配置 **图像**、**视频** 模型（文生视频类计划需要）
2. 在 **Agent Tab → 工作流模板** 中按需开关内置模板
3. 打开或创建一个项目

### 基本流程

1. 打开 **Agent 面板**（右下角 🤖 浮岛，或钉住侧栏）
2. 输入自然语言意图，或 **采纳推荐/快速模板**，例如：
   - 「做一个 15 秒咖啡品牌宣传片」
   - 「用首尾帧做从日出到城市夜景的过渡视频」
   - 「根据这个故事写分镜并最后合成导出」
3. 阅读回复中的 **工作流计划预览**（节点列表 + 摘要）
4. 点击 **确认并添加到画布** — 节点与连线会出现在当前视口附近
5. 若计划为 **自动模式**（`executionMode: auto`），确认后会立即触发 **DAG 整组执行**；手动模式需你自行在各节点点击生成或输入 `/run`

### 计划中的警告

Agent 回复或摘要末尾可能出现 `⚠` 警告，常见原因：

| 警告 | 含义 |
|------|------|
| 未找到已接入的 xx 模型 | 缺少对应类型模型，需先在设置中添加 |
| 无完全匹配能力的模型，已选最接近的… | 没有满足尾帧/Vision/参考图等要求的模型，已回退到最接近项 |
| 按能力需求优选 | 默认模型不满足连线推断的能力需求，已改用更合适的模型 |

摘要中会附带 **已选模** 列表（如「已选模：DeepSeek V4 Flash、Seedance 2.0」）。

## Agent 如何生成计划

请求按以下优先级处理：

```
用户输入
  ├─ 1. 关键词命中内置模板 → 展示 Top 模板卡片（需用户「采纳」）
  ├─ 2. 用户跳过模板 + 已配置默认 LLM → LLM 工作流规划器输出 JSON
  └─ 3. LLM 失败 → 回退到「文生图生视频」模板（若未在设置中关闭）
```

计划预览若出现 **blocking** 警告（缺模型），主按钮为 **去配置**，跳转 **设置 → Agent** 就绪卡。

无论哪条路径，计划都会经过 **能力选模**（`enrichWorkflowPlanWithModels`）：根据每个节点的类型与入边推断需求，从已接入模型中打分选择 `modelId`。

### 能力选模规则（摘要）

| 节点 | 推断依据 | 选模要点 |
|------|----------|----------|
| 文本 | 是否有图片连到 `image` / `imageN` | 需要 Vision 时选支持图输入的 LLM |
| 图像 | 是否有连到 `reference` | 需要参考图时选 Seedream 4.5 等 |
| 视频 | `firstFrame` / `lastFrame` / `referenceN` / 参考音视频 | 首尾帧过渡 **必须** 支持尾帧（如 Seedance 2.0） |

评分会优先：默认模型 → 能力置信度（verified / documented）→ 内置目录 → 是否满足特殊需求（如尾帧）。

LLM 规划时，系统会把 **已接入模型能力目录** 注入系统提示，规则包括：

- 仅需文案 → 选无图输入 LLM
- 需要 vision → 选支持图输入 LLM
- 首尾帧过渡 → 必须选支持尾帧的视频模型

## 内置技能

在 **设置 → 应用设置 → Agent 预置技能** 可勾选禁用单项技能。禁用后关键词不再匹配，LLM 失败时也不会回退到该技能。

| 技能 ID | 名称 | 触发词（部分） | 执行模式 | 生成内容 |
|---------|------|----------------|----------|----------|
| `text-to-video` | 文生图生视频 | 文生视频、宣传片、广告、短视频… | **自动** | 文本 → 图像 → 视频（含文本到视频 prompt 边） |
| `first-last-frame` | 首尾帧过渡 | 首尾帧、过渡、插帧… | **自动** | 双文本 + 双图像 + 视频（firstFrame + lastFrame） |
| `script-to-film` | 脚本成片 | 脚本、分镜、故事、短片… | **手动** | 脚本节点 + 合成节点骨架 |

**自动模式**：确认计划后立即 DAG 执行计划内全部可生成节点（`text` / `image` / `video`）。

**手动模式**：仅落盘节点结构；你需在脚本节点生成分镜、批量出图/出视频，再在合成剪辑台完成导出。

## 工作流计划结构

计划 JSON（内部格式，用户无需手写）包含：

| 字段 | 说明 |
|------|------|
| `summary` | 一句话摘要；选模后会追加「已选模：…」 |
| `nodes[]` | `tempId`、`type`、`label`、`data`（含 `modelId`）、可选 `position` |
| `edges[]` | `sourceHandle` / `targetHandle` 与画布端口一致 |
| `executionMode` | `auto` 或 `manual` |
| `estimatedSteps` | 预估步骤数 |
| `skillId` | 若来自内置技能则标注 |

支持的节点类型：`text`、`image`、`video`、`audio`、`script`、`compose`、`storyboard`。

## DAG 批量执行

Agent 自动模式、斜杠命令与右键菜单共用同一套 DAG 引擎。

### 触发方式

| 方式 | 作用范围 |
|------|----------|
| Agent 确认计划（`auto` 模式） | 计划内全部节点 |
| `/run` | 当前 **选中节点**；无选中时 **全画布** |
| 右键 → **整组执行** | 当前选中节点 |

### 执行逻辑

1. 收集子图并按拓扑序排序（有环则报错）
2. 仅对 `text` / `image` / `video` 节点触发生成；`script`、`compose` 等跳过
3. 执行前应用数据流连线补丁（prompt、首帧图等）
4. 文本节点生成时会收集 Vision 入边图片
5. 底部 **DAG 运行面板** 显示 pending / running / completed / failed

生成前会校验关键入边能力；不兼容的虚线连线可能导致该节点 failed。

## 斜杠命令

在画布按 `/` 打开命令面板（输入框、文本节点编辑时不会触发）：

| 指令 | 说明 |
|------|------|
| `/run` | 对选中节点（或全画布）按拓扑序 DAG 执行 |
| `/agent` | 打开 Agent 面板 |
| `/grid 3x3` | 将选中（或全部）分镜组切换为九宫格 |
| `/grid 5x5` | 将选中（或全部）分镜组切换为二十五宫格 |
| `/export storyboard` | 导出第一个分镜组为 **PNG 拼图**（按当前 layout：列表/九宫格/二十五宫格），并打开输出目录 |
| `/style` | 打开 Generator Drawer 并滚动高亮 **风格** chip 行（图片节点） |

## 分镜组

分镜组（`storyboard` 节点）常与 Agent「脚本成片」或脚本节点配合使用。

- **脚本节点** 右键 → **转为分镜组**
- 分镜组生成器面板可切换 **列表 / 九宫格 / 二十五宫格**（也可用 `/grid`）；**画布节点预览**与所选 layout 同步
- 勾选帧后可 **重生成选中帧**（按帧状态自动重生成图片或视频）
- 支持导出 **PNG 拼图**（编辑器侧栏或 `/export storyboard`），或导出选中帧 **4K** 单图；**不再支持 PDF 导出**（v9 已移除）
- 「**同步到画布**」可为每帧创建/更新关联图片节点

## 合成与字幕

Agent「脚本成片」会预置 **合成节点**。选中合成节点后打开 **合成剪辑台**：

1. 将视频片段连线到 `video1` / `video2` … 或在时间线上排列
2. 工具栏 **字幕** → 导入 **SRT** → 时间轴显示字幕轨，预览窗口同步显示
3. 导出抽屉可勾选 **烧录硬字幕**，一并写入 MP4

## 音频后期

- 音频节点生成或导入后，生成器面板可 **分离人声**
- **设置 → 应用设置** 配置 `Demucs` 路径可获得更好分离效果
- 未配置时回退 FFmpeg 简易分离；完成后自动创建「人声」「伴奏」子节点

## 风格模板

图像生成器（**ImageEditorPanel**）使用 **风格 chip**（电影感、动漫、产品广告等）：选中后即时显示「生效 prompt」与负向词预览；生成时仍通过 `applyStyleToPrompt` 拼 prefix，**不**改写节点上的用户原文 prompt。

- `/style` — 打开底部 Drawer 并 focus 风格行（需已选中图片节点）
- 部分风格含 `recommendedImageModel`：面板会提示「推荐 xxx · 应用」一键切换模型
- 画布图片节点标题旁 **badge** 显示当前风格名
- 视频生成器仍为下拉选择，chip UI 待与图片对齐

## 会话与账号

- Agent 对话会写入本地 `agent_sessions` 表（按 `user_id` / `project_id` 隔离）
- 首次启动可选 **游客继续** 或注册本地账号；游客模式功能不受限
- API Key 仅存本机，Agent 仅将意图发送给你配置的 LLM 端点

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| Agent 只回复文字、无计划 | 检查默认 LLM 是否配置；或换用含技能触发词的描述 |
| 计划里视频模型不对 | 首尾帧需 Seedance 2.0；在设置中添加并设为默认视频模型 |
| 确认后 DAG 未运行 | 该计划为 `manual` 模式（如脚本成片）；手动 `/run` 或逐节点生成 |
| 自动执行某节点失败 | 查看 DAG 面板错误；检查虚线连线与 API Key |
| 想禁用某个技能 | 设置 → 应用设置 → Agent 预置技能，取消勾选 |
