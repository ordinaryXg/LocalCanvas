# LocalCanvas 账号说明

> Wave D · T10-EXE-05

## 游客模式

- 首次打开可使用**游客身份**进入，无需注册。
- 游客数据保存在本机，卸载应用或清除数据会丢失账号信息。
- 可在右上角账号菜单中**注册**或**登录**，将游客数据绑定到正式账号（若服务端支持）。

## 昵称与头像

- 登录后可在**账号菜单 → 个人资料**修改显示昵称。
- 头像目前为本地占位，云端头像同步为远期能力。

## 数据与隐私

- 项目文件、生成资产默认保存在本机用户目录。
- API Key 写入本地 `config.yaml`，不会上传到 LocalCanvas 官方服务器。
- 模型调用直连您配置的火山方舟 / 其他服务商 endpoint。

## 常见问题

| 问题 | 说明 |
|------|------|
| 忘记密码 | 使用登录页的找回流程（需已配置认证服务） |
| 换电脑 | 导出项目文件夹或手动复制 `LocalCanvas` 数据目录 |
| 多设备同步 | 当前版本不支持，见产品路线图 |

更多模型与能力配置，请参阅应用内 **⚙️ 设置 → 已接入模型**。
