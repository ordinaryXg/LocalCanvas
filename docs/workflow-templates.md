# 工作流模板

LocalCanvas 内置 4 个预置工作流，可在侧边栏 **工具 → 工作流模板** 中一键加载到画布。加载时画布会自动偏移位置，避免与现有节点重叠。

也可通过侧边栏 **Agent** 用自然语言生成类似结构的工作流；Agent 会按能力目录自动选择模型并在计划中标注 `modelHint`。

## 预置模板

### 1. 文生图 → 图生视频

**用途**：从文字描述生成静态图，再基于首帧生成短视频。

**节点**：文本 → 图像 → 视频

**连线**：

- 文本 `prompt` → 图像 `prompt`
- 图像 `image` → 视频 `firstFrame`

**建议步骤**：

1. 在文本节点填写画面描述（草稿栏或直接写入）
2. 选中图像节点，在生成面板选择图像模型并生成
3. 选中视频节点，选择 Seedance 模型，配置时长与画幅后生成
4. 若需运动描述，可将文本 `prompt` 额外连到视频 `prompt`

**推荐模型**：Seedream 4.5（图）+ Seedance 1.0 Pro Fast 或 2.0（视频）

---

### 2. 脚本 → 分镜 → 批量视频

**用途**：输入故事梗概，由 LLM 生成分镜脚本，再批量出图/出视频。

**节点**：脚本节点（单节点起步，可在画布上扩展）

**建议步骤**：

1. 在脚本节点填写故事与角色设定
2. 配置默认 LLM 后使用「生成分镜」生成表格
3. 通过脚本面板批量生成图片或视频
4. 可将分镜组打组，用 `/grid 3x3` 或 `/grid 5x5` 切换宫格布局
5. 用 `/export storyboard` 导出分镜 PNG 拼图

**Agent 等效技能**：「脚本成片」— 会创建脚本节点与合成节点骨架，后续步骤需手动批量生成

---

### 3. 首尾帧视频生成

**用途**：分别生成起始帧与结束帧，再生成过渡视频。

**节点**：文本（起始）+ 文本（结束）→ 图像 × 2 → 视频

**连线**：

- 起始文本 `prompt` → 起始图像 `prompt`
- 结束文本 `prompt` → 结束图像 `prompt`
- 起始图像 `image` → 视频 `firstFrame`
- 结束图像 `image` → 视频 `lastFrame`

**重要**：尾帧端口需要 **Seedance 2.0** 系列模型。使用 Seedance 1.0 时，连到 `lastFrame` 的边会显示虚线，生成时会被拦截。

**可选扩展**：

- 将更多参考图连到 `reference1` … `reference9`（Seedance 2.0，最多 9 张）
- 用 Agent 技能「首尾帧过渡」可一键生成上述节点结构

---

### 4. 多片段合成导出

**用途**：将多个视频片段按时间线拼接，可选混入背景音乐。

**节点**：视频 × 3 + 音频 + 合成

**连线**：

- 视频 `video` → 合成 `video1` / `video2` / `video3`
- 音频 `audio` → 合成 `audio`

**建议步骤**：

1. 在各视频节点生成或导入片段
2. **选中合成节点** — 自动打开 **合成剪辑台**（时间线 + 预览 + 检查器）
3. 在时间线上调整顺序、入点/出点、音量；可选导入 SRT 字幕
4. 从导出抽屉配置分辨率与编码，输出 MP4 到 `output_dir`（默认 `~/LocalCanvas/outputs`）

> 合成节点不再使用旧版简单列表面板，统一走全屏剪辑台体验。

---

## 自定义工作流

1. 在画布上搭建节点并连线（注意虚线表示当前模型不兼容）
2. 选中节点组后 `Ctrl+G` 打组
3. 右键 **保存为工作流**（或通过项目菜单导出）
4. 在 **我的工作流** 列表中加载、导出或删除

导入/导出使用 JSON 格式，可在团队间共享。用户保存的工作流可删除；内置预置不可删除。

## 批量执行

加载模板并完成内容填写后：

1. 选中要执行的节点（或整组）
2. 输入 `/run` 或从菜单触发 **整组执行**
3. 底部 DAG 面板按拓扑顺序依次触发生成，展示各节点状态

依赖未就绪的节点会标记为 skipped；能力不匹配的节点会 failed 并显示原因。

## Agent 与工作流的关系

| Agent 技能 | 对应模板思路 | 执行模式 |
|------------|--------------|----------|
| 文生图生视频 | 模板 1 | 自动（`executionMode: auto`） |
| 首尾帧过渡 | 模板 3 | 自动 |
| 脚本成片 | 模板 2 + 合成 | 手动（需用户批量生成） |

在 **设置 → 应用设置** 可单独禁用某个 Agent 技能。

## 开发者备注（IPC）

| 通道 | 说明 |
|------|------|
| `workflow:list` | 列出预置与用户工作流 |
| `workflow:load` | 加载到画布（自动偏移避免重叠） |
| `workflow:save` | 保存当前选区 |
| `workflow:delete` | 删除用户工作流 |
| `workflow:import` / `workflow:export` | 文件导入导出 |
