# 工作流模板

LocalCanvas 内置 4 个预置工作流，可在侧边栏 **工具 → 工作流模板** 中一键加载到画布。

## 预置模板

### 1. 文生图 → 图生视频

**用途**：从文字描述生成静态图，再基于首帧生成短视频。

**节点**：文本 → 图像 → 视频

**连线**：

- 文本 `prompt` → 图像 `prompt`
- 图像 `firstFrame` → 视频 `firstFrame`

**建议步骤**：

1. 在文本节点填写画面描述
2. 选中图像节点，在生成面板选择图像模型并生成
3. 选中视频节点，配置时长与分辨率后生成

---

### 2. 脚本 → 分镜 → 批量视频

**用途**：输入故事梗概，由 LLM 生成分镜脚本，再批量出图/出视频。

**节点**：脚本节点

**建议步骤**：

1. 在脚本节点填写故事与角色设定
2. 使用「生成分镜」生成表格
3. 通过脚本面板批量生成图片或视频

---

### 3. 首尾帧视频生成

**用途**：分别生成起始帧与结束帧，再生成过渡视频。

**节点**：文本（起始）+ 文本（结束）→ 图像 × 2 → 视频

**连线**：

- 起始文本 → 起始图像 → 视频 `firstFrame`
- 结束文本 → 结束图像 → 视频 `lastFrame`

---

### 4. 多片段合成导出

**用途**：将多个视频片段按时间线拼接，可选混入背景音乐。

**节点**：视频 × 3 + 音频 + 合成

**建议步骤**：

1. 在各视频节点生成或导入片段
2. 在合成节点配置片段顺序与时长
3. 可选连接音频节点作为 BGM
4. 执行合成并导出到 `LocalCanvas/outputs`

---

## 自定义工作流

1. 在画布上搭建节点并连线
2. 选中节点组后右键 **保存为工作流**（或通过项目菜单导出）
3. 在 **我的工作流** 列表中加载、导出或删除

导入/导出使用 JSON 格式，可在团队间共享。

## 相关 IPC

| 通道 | 说明 |
|------|------|
| `workflow:list` | 列出预置与用户工作流 |
| `workflow:load` | 加载到画布（ToolPanel 会偏移位置避免重叠） |
| `workflow:save` | 保存当前选区 |
| `workflow:delete` | 删除用户工作流（预置不可删） |
| `workflow:import` / `workflow:export` | 文件导入导出 |
