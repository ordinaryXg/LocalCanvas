# LocalCanvas Agent 使用指南

## 快速开始

1. 在画布按 `/` 打开指令面板，输入 `/agent` 打开 **Agent 面板**。
2. 用自然语言描述目标，例如：「做一个 15 秒咖啡品牌宣传片，先写脚本再出分镜图」。
3. 查看 **工作流计划预览**，确认后点击「确认并添加到画布」。
4. 若计划为自动模式，将触发 **DAG 整组执行**；也可在画布右键选择「整组执行」或输入 `/run`。

## Slash 指令

| 指令 | 说明 |
|------|------|
| `/run` | 对当前选区（或全画布）按拓扑序执行 |
| `/agent` | 打开 Agent 面板 |
| `/grid 3x3` | 分镜组切换九宫格 |
| `/grid 5x5` | 分镜组切换二十五宫格 |
| `/export storyboard` | 导出选中分镜组 PNG 拼图 |
| `/style` | 在图像/视频生成器使用风格模板 |

## 分镜组

- 脚本节点右键 → **转为分镜组**
- 勾选帧后可批量重生成图片
- 生成器面板可导出 **PNG / PDF**，或导出单帧 **4K**

## 音频后期

- 音频节点生成或导入后，生成器面板可 **分离人声**
- 设置 → 系统：配置 `Demucs` 路径可获得更好效果；未配置时使用 FFmpeg 简易分离
- 分离后自动创建「人声」「伴奏」两个子节点

## 字幕

- 合成节点 → **导入 SRT** → 时间轴预览字幕轨
- 勾选 **烧录硬字幕** 后导出合成视频

## 风格模板

图像/视频生成器中选择风格（电影感、动漫、产品广告等），会自动追加提示词前缀与负向词。

## 技能开关

设置 → Agent：可关闭不需要的预置 Skill（文生视频、脚本成片、首尾帧等）。

## 账号与数据

- 支持本地注册/登录；游客模式可跳过
- 项目与历史按 `user_id` 隔离，为 v6 云端迁移预留字段
