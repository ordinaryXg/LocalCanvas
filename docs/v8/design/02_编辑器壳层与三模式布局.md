## 一、EditorShell 结构

```tsx
export function EditorShell() {
  const mode = useEditorMode(s => s.mode)
  return (
    <div className="editor-shell w-screen h-screen flex flex-col bg-[var(--studio-bg)]">
      <TopBar />
      <div className="flex-1 min-h-0 flex">
        <Dock />
        <main className="flex-1 min-w-0 relative">
          {mode === 'canvas' && <CanvasMode />}
          {mode === 'generate' && <GenerateMode />}
          {mode === 'edit' && <EditMode />}
        </main>
        {mode !== 'edit' && <Inspector />}
      </div>
      <GeneratorDrawer />
      <AgentCompanion />
    </div>
  )
}
```

## 二、CanvasMode

### 2.1 布局

```
┌────────────────────────────────────┐
│  Canvas (React Flow)               │
│  · CanvasToolbar (左上)            │
│  · MiniMap (右下, 设置可关)         │
│  · 缩放指示 (左下)                  │
│  · ContextMenu / SlashPalette      │
└────────────────────────────────────┘
```

### 2.2 与 v6 差异

| v6 | v8 |
|----|-----|
| `GeneratorPanel` 居中浮层 | 移除；改 `GeneratorDrawer` |
| 选中合成节点自动弹 75% ComposeEditor | **不自动**；Inspector 显示「打开剪辑台」→ 切 edit 模式 |
| `Sidebar` 5 Tab | `Dock` + 抽屉 |

### 2.3 保留不变

- 节点类型、连线、`isValidConnection`、Capability 警告样式
- `DagRunPanel`、`SlashCommandPalette`、文件拖入

## 三、GenerateMode

### 3.1 进入条件

- 用户按 `G` 或 Inspector 点击「生成」
- 当前必须选中 **单个** 可生成节点

### 3.2 布局

```
┌──────────────────────────────┬─────────────┐
│  预览区 (节点大图/视频预览)    │  参数表单    │
│  底部: GenerationHistoryStrip │  模型选择    │
│  (横向滚动该节点历史)          │  生成按钮    │
└──────────────────────────────┴─────────────┘
```

- 右栏复用现有 Generator 表单组件（抽 `GeneratorForm`）。
- Esc → canvas 模式。

---

## 四、EditMode

### 4.1 进入条件

- Inspector「打开剪辑台」
- 或 ModeSwitcher 手动选「剪辑」（若有选中合成节点）

### 4.2 布局

- 全宽 `ComposeEditor`，`variant="shell"`：
  - 隐藏 ComposeEditor 自有顶栏的项目级操作，改用 Shell TopBar
  - 保留：预览、时间轴、检查器、导出
- 顶栏追加「返回画布」→ `setMode('canvas')`

### 4.3 高度

`localStorage.composeHeightRatio`：默认 0.85（edit 模式几乎全屏）；canvas 模式内嵌预览若未来需要另议。

## 五、Dock 抽屉

```typescript
type DockDrawer = 'nodes' | 'tools' | 'assets' | 'history' | null

interface DockState {
  openDrawer: DockDrawer
  drawerWidth: number // default 320
}
```

**互斥规则**：打开新 drawer 关闭旧 drawer；再次点击同一图标 → 关闭。

## 六、Inspector 内容矩阵

| 选中 | 区块 |
|------|------|
| text | 标题 · outputMode chip · 字数 · Vision 计数 · 连线 · 「打开文本编辑器」 |
| image/video/audio | 48px 状态图 · 状态 chip · 摘要（来源/字数）· 连线 · 健康摘要 · 高级（画布尺寸）· 「打开编辑器」 |
| compose | 片段/时长 chip · 上次导出 · 「打开剪辑台」 |
| script | 行数/出图计数 chip · 梗概摘要 · 「打开脚本编辑器」 |
| storyboard | 帧数/layout chip · 只读 3×3 概览 · 「打开分镜编辑器」 |
| edge | from→to · 能力状态 · 「断开」 |
| multi | 选中数 · 类型分布 · 整组执行 · 删除 |
| null | 项目名 · 节点/边计数 · 「+ 文本节点」 |
