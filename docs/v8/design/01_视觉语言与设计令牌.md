## 一、设计气质

**Studio Dark** — 本地创作工具：暗色、低干扰、信息分层清晰。

| 是 | 不是 |
|----|------|
| 大留白、细边框、轻阴影 | 密铺卡片、高对比分隔 |
| indigo 作主 accent | 彩虹状态色 |
| 动效短、有目的 | 装饰性 bounce |
| 14px 正文 + 清晰标题级 | 全站同一字号 |

## 二、色板扩展（`theme.css`）

在 v6 已有 `--color-bg-primary`、`--color-text-*` 等基础上，新增 **`studio-*` 命名空间**。

```css
:root[data-theme='dark'] {
  /* 背景层级 */
  --studio-bg: #0f1117;
  --studio-surface: rgba(255, 255, 255, 0.03);
  --studio-surface-hover: rgba(255, 255, 255, 0.06);
  --studio-border: rgba(255, 255, 255, 0.08);

  /* 强调 */
  --studio-accent: #6366f1;
  --studio-accent-muted: rgba(99, 102, 241, 0.25);

  /* 三模式（ModeSwitcher） */
  --mode-canvas: #6366f1;
  --mode-generate: #f59e0b;
  --mode-edit: #64748b;

  /* 状态 */
  --status-running: #38bdf8;
  --status-success: #4ade80;
  --status-error: #f87171;
  --status-warning: #fbbf24;
}
```

与 v6 关系：`--studio-bg` 初始值对齐 `--color-bg-primary`；迁移期两套并存，组件逐步改用 `studio-*`。

## 三、组件样式约定

### 3.1 ModeSwitcher 胶囊

```
容器: rounded-full bg-zinc-800/80 p-1 gap-0.5
激活: bg-[var(--mode-*)] text-white text-sm font-medium px-3 py-1
未激活: text-zinc-400 hover:text-zinc-200 px-3 py-1
```

### 3.2 Dock

```
列宽: 56px (--space-dock)
图标: 24px, 默认 text-zinc-500
激活: 左侧 2px var(--studio-accent) 条 + text-white
抽屉: width 320px 默认, bg-bg-secondary, border-r border-border
```

### 3.3 Inspector

```
宽: 280px 默认, 可拖 240–360px
分区标题: text-xs uppercase tracking-wide text-zinc-500 mb-2
字段行: py-2 border-b border-studio-border last:border-0
空状态: 居中 illustration + 「选中节点以查看详情」
```

### 3.4 GeneratorDrawer

```
顶栏: h-10 flex items-center border-b border-studio-border px-4
拖动手柄: 顶边 h-2 cursor-ns-resize hover:bg-studio-accent-muted
内容区: overflow-y-auto lc-scroll px-4 py-3
```

### 3.5 AgentCompanion

```
FAB: 56×56 rounded-full shadow-lg bg-zinc-800 border border-studio-border
卡片: w-80 rounded-xl border border-studio-border bg-bg-secondary/95 backdrop-blur
钉住态: 贴右侧全高, 宽 320px, 与 Inspector 互斥
```
