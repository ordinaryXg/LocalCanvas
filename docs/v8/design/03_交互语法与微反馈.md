## 一、全局快捷键

| 键 | 上下文 | 动作 |
|----|--------|------|
| `G` | 选中可生成节点、非输入 focus | 打开 GeneratorDrawer 或进入 generate 模式 |
| `E` | 选中合成节点 | 进入 edit 模式 |
| `Esc` | generate/edit 模式 | 回 canvas 模式 |
| `Esc` | Drawer 开 | 关 Drawer |
| `Esc` | Inspector mobile overlay | 关 overlay |
| `Ctrl+S` | 全局 | 手动保存 |
| `?` | 全局 | 快捷键 cheatsheet |
| `Delete` | 画布 focus | 删除选中（与 v6 一致） |
| `/` | 画布 | SlashCommandPalette（与 v6 一致） |

> 输入框 / textarea focus 时，`G` / `E` / 模式键不触发。

## 二、GeneratorDrawer

### 2.1 状态

```typescript
interface GeneratorDrawerState {
  open: boolean
  heightRatio: number   // 0.4 default, clamp 0.25–0.7
  nodeId: string | null
}
```

### 2.2 打开/关闭

| 事件 | 行为 |
|------|------|
| 选中可生成节点 + 双击 | 打开 |
| Inspector「生成」 | 打开 |
| `G` | 打开 |
| 点击遮罩（画布区） | 关闭（生成进行中除外） |
| 切换选中到其他节点 | 更新 nodeId，保持打开 |

### 2.3 动效

- `transform: translateY(100%) → 0`，250ms `ease-out`
- 顶边 drag handle 调整 `heightRatio`

### 2.4 生成中

```
[████████░░░░] 42%  正在生成图片…  [取消]
```

- 取消：调用现有 abort（若 adapter 支持）
- 允许用户收起 Drawer（进度缩到 TopBar 小 Badge）

## 三、AgentCompanion

### 3.1 三态

```
collapsed (FAB) → expanded (card) → pinned (右侧 320px 列)
```

| 状态 | 尺寸 |
|------|------|
| collapsed | 56px FAB |
| expanded | 320×min(480px, 70vh) 卡片，距右下 16px |
| pinned | 右侧固定，Inspector 自动折叠 |

### 3.2 WorkflowPlan

- 默认：**折叠块** + 标题「Agent 规划了 N 步」+ 「展开」
- 展开后：内嵌滚动，最大高度 240px
- **不**使用 v6 全屏 `WorkflowPlanPreview`（除非用户点击「全屏查看」）

### 3.3 建议 chip

Agent 返回可执行建议时，渲染为可点击 chip：

- 「为选中图片节点生成」→ 打开 Drawer + 预填
- 「添加合成节点」→ canvas 模式 + 创建节点

## 四、保存与 Toast

| 事件 | 反馈 |
|------|------|
| 自动保存成功 | TopBar `· 已保存`（3s 后淡化） |
| 自动保存失败 | Toast error + 保持「未保存」 |
| 手动 Ctrl+S | TopBar spinner → 已保存 |
| 生成成功 | Toast info 3s + Inspector 刷新预览 |
| 生成失败 | Toast error 5s + Drawer 内错误详情 |

## 八、降级与经典布局

| 设置项 | 效果 |
|--------|------|
| 使用经典布局 | 渲染 `LegacyAppLayout`，行为与 v6 完全一致 |
| 关闭 Coach | 跳过引导 |
| 默认关闭 MiniMap | CanvasMode 不渲染 MiniMap |

经典布局路径 **不删除**，作为长期 escape hatch。
