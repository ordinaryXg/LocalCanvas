# LocalCanvas v2 — 模型配置 + 生成器系统

> **版本目标**：远端 API 全链路生成（文本/图/视频/音频），画布「可产出」  
> **周期**：2.5 周 · **状态**：✅ 已验收  
> **详案**：[v2/LocalCanvas_v2_模型配置与生成器系统.md](./v2/LocalCanvas_v2_模型配置与生成器系统.md)

---

## 核心交付

| 模块 | 要点 |
|------|------|
| 配置 | `config.yaml`、设置页添加模型、热重载 |
| 适配器 | RemoteApi、Seedance 2.0（火山方舟）为核心视频引擎 |
| 生成器 | 文本/图/视频/音频面板、TaskQueue、进度推送 |
| 脚本 | 脚本节点表格、批量分镜生成 |

## 验收摘要

- LLM / 图像 / 视频 / TTS 可配置并生成
- Utility Process 执行、错误分类、脚本批量链路可用
