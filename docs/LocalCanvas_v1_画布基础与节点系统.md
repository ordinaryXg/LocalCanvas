# LocalCanvas v1 — 画布基础 + 节点系统

> **版本目标**：无限画布、五种节点、连线打组、项目本地存取  
> **周期**：2 周 · **状态**：✅ 已验收（2026-06-04）  
> **详案**：[v1/LocalCanvas_v1_画布基础与节点系统.md](./v1/LocalCanvas_v1_画布基础与节点系统.md)

---

## 核心交付

| 模块 | 要点 |
|------|------|
| 应用骨架 | Electron + IPC + SQLite 项目存取 |
| 画布 | React Flow、缩放/平移/小地图、撤销重做 |
| 节点 | 文本 / 图片 / 视频 / 音频 / 脚本 + 端口连线 |
| 交互 | 打组、右键菜单、双击创建、文件拖入 |
| 主题 | 深色 CSS 变量 + Tailwind |

## 验收摘要

- 5 种节点创建与合法连线、数据传递、自动保存
- 单测覆盖率 ≥80%、错误处理框架就绪
- 视频内联播放已实现；首帧缩略图留 v3
