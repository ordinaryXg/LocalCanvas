# LocalCanvas v7 — 液态创作（零约束愿景）

> **本文档不受任何前置版本、技术栈、节点范式或发布周期的约束。**  
> 它只回答一个问题：**如果 AI 是创作的主体之一，而不是工具，创作环境应该长什么样？**  
> **生成日期**：2026-06-06  
> **立场**：这不是「v6 的下一补丁」，而是一份**可证伪的激进假设集**——你可以全部否定，但至少它不再假装保守。

---

## 〇、先拆掉什么

在构思 v7 之前，先主动卸载这些默认假设：

| 被卸载的假设 | 为什么可疑 |
|--------------|------------|
| 画布 + 节点是正确隐喻 | 节点是**编译器 IR**，不是人类思维的形式 |
| 用户先规划再生成 | 创作是**螺旋收敛**，不是瀑布流 |
| 视频是最终产物 | 视频是**一种冻结方式**；活态叙事才是本体 |
| Agent = 聊天框里的助手 | 助手暗示从属；**共生**才接近真实协作 |
| 本地 = 数据在硬盘 | 本地应是**意识主权**——你的审美史不被平台吞噬 |
| 一致性 = 角色脸不变 | 一致性也可以是**情绪逻辑、隐喻系统、节奏呼吸** |
| 分支 = Git | Git 为工程师设计；创作者需要的是**平行梦境** |
| 质量 = 生成后评审 | 真正稀缺的是**生成前的共鸣**与**生成中的共感** |

**v7 不是「更好的节点编辑器」。**  
如果实现后用户仍说「这是个连线的 AI 工具」，那 v7 就失败了。

---

## 一、核心隐喻：创作是液态的

### 1.1 流体，不是图

想象创作物不是 DAG，而是一池**始终流动的液体**：

- **温度** = 确定性（冷 = 已敲定，热 = 仍在沸腾）
- **粘度** = 改动成本（低粘度区域轻轻一推就变形）
- **表面张力** = 不同场景/角色之间的边界
- **结晶** = 你决定「这里不再动了」的瞬间——导出视频，只是大规模结晶

用户不是在「摆放节点」，而是在**调节流体的物理参数**，偶尔伸手掬起一捧，看它是否是你想要的形状。

### 1.2 呼吸，不是执行

人与 AI 的协作节奏应像**共同呼吸**：

```
吸气  →  你给出模糊感受、碎片、哼唱、涂鸦、一句不对的话
屏息  →  AI 不急着输出，在内部展开多种可能（用户看不见）
呼气  →  共同看见一个「足够真」的瞬间
```

现在的产品大多在「吸气」刚结束就「呕吐式输出」一整张工作流。  
v7 的核心体验是：**学会一起屏息。**

## 二、七个激进概念

### 2.1 共鸣场（Resonance Field）

**取代：参考图 + Prompt 工程**

用户不再写「赛博朋克、霓虹、雨夜、8k、大师级」。  
用户给出**共振源**——可以是：

- 一段没歌词的哼唱
- 一张模糊到几乎看不清的照片
- 一句让你心悸的话
- 另一个作品的**3 秒钟**（不是整片）

AI 的任务不是「模仿」，而是提取**共振频率**：色温曲线、节奏密度、隐喻核、情感导数。  
所有后续生成都在这个频率附近**调谐**，而不是在 prompt 字符串附近拼接。

**UI 形态**：不是输入框，是一个**调谐盘**——多个共振源像行星一样轨道运行，你可以拉近/推远、加强/减弱某一颗的引力。

---

### 2.2 叠加态镜头（Superposed Shot）

**取代：生成 → 不满意 → 重 roll**

每一个镜头在未被「观测」（敲定）之前，**同时存在多个版本**。  
不是分支树的兄弟节点，而是**量子叠加**：

-  scrub 时间轴时，看到的是概率云——主版本最亮，备选版本像鬼影
- 鼠标悬停某帧 → 叠加态展开成扇形胶片
- 「观测」= 你选择这一刻坍缩成哪一个现实

这改变了心理模型：  
创作者不是在「失败重试」，而是在**选择哪一个宇宙留下来**。

---

### 2.3 情感地形（Affect Terrain）

**取代：分镜表 + 情绪标签**

把整部作品的情感历程画成**可行走的三维地形**：

- 横轴：时间
- 纵轴：唤醒度（平静 ↔ 激烈）
- 深度：效价（阴郁 ↔ 明亮）

每个镜头是一个**地形上的巨石或洼地**。  
你拖动时间指针行走，AI 根据你脚下的高度差自动建议：「这里下坡太陡，要加缓冲镜头吗？」

观众尚未看见成片，但**情感物理已经可模拟**。  
这是生成前的共鸣——不是生成后的打分。

---

### 2.4 隐迹记忆（Palimpsest Memory）

**取代：角色圣经 + 固定 reference**

创作记忆不应是「锁死的设定集」，而是**羊皮纸上的多层字迹**：

- 第 1 稿的主角穿红夹克——后来被否了，但「红」作为隐喻残留
- 第 7 次迭代里某个被拒绝的眼神，可能在第 20 镜以另一种形式复活

AI 维护的不是「事实表」，而是**创伤与执念的考古层**。  
当你说「再忧郁一点」，系统知道你说的是**哪一层忧郁**——是第 3 次迭代的那种，还是最初 brief 里那种。

记忆是**有深度的**，不是扁平的 key-value。

---

### 2.5 内在合唱团（Inner Chorus）

**取代：导演/摄影/剪辑多 Agent 角色扮演**

六个职业 Agent 仍是人类剧组的投影，仍带着工业时代的分工幻觉。

更激进的做法：**合唱团是你自己的多重自我**——

| 声部 | 象征 |
|------|------|
| 冲动者 | 「我就要现在爆炸」 |
| 怀疑者 | 「这太俗了」 |
| 怀旧者 | 「像小时候那部电影」 |
| 完美主义者 | 「这里手型不对」 |
| 旁观者 | 「观众会在这里走神」 |
| 沉睡者 | 尚未成形、只可隐约感觉的方向 |

合议不是六个职业人士开会，而是**你脑子里的六个声音被外化了**。  
AI 不替你决定，它让声音**被听见**，然后你拍板——或选择「今天让怀疑者当家」。

---

### 2.6 负熵创作（Negentropic Creation）

**取代：从 prompt 正向堆叠**

大多数坏创意是**加太多**。  
v7 提供一种反向模式：

1. 从混沌开始——噪声图、随机剪辑、无意义句群
2. 用户说「去掉让我感到假的部分」
3. AI 减法迭代，而非加法迭代
4. 直到剩下**无法再继续削减**的核

这是雕塑式创作。对 AI 时代「生成泛滥」的一种反叛。  
**删除**应和**生成**一样是一等按钮。

---

### 2.7 共生预演（Symbiotic Latency）

**取代：点击生成 → 等待 → 查看**

AI 在你编辑意图的**同时**，已在背景低分辨率预演多种可能。  
不是偷偷浪费算力，而是**共享注意力**：

- 你改一个词，预演池轻微偏转——像磁铁靠近铁屑
- 你停顿 3 秒，最可能的方向以「幽灵预览」浮现画布边缘
- 你点头，才升格为正式渲染

用户感觉不是「我在命令机器」，而是**机器在读我的犹豫**。

## 三、一个完整的创作会话（叙事原型）

> 没有节点。没有 DAG。只有体验描述。

---

**早晨**  
你打开项目。屏幕中央不是空白画布，而是一片**微热的雾**——上次离开时作品的「流体余温」。

你哼了 8 秒钟，扔进去一段地铁窗倒影的模糊视频。  
调谐盘上多了一颗暗蓝色行星。雾开始朝那个颜色靠拢。

---

**上午**  
你没有写分镜。你在**情感地形**上用手指划了一道缓坡：前 30 秒要「闷」，然后突然坠落。  
地形上出现三处 AI 建议的「悬崖」位置。你选了中间那个。

---

**午后**  
第一个「镜头」以叠加态出现——4 个鬼影版本同时呼吸。  
你删掉两个（负熵），让一个坍缩，剩两个保持叠加。  
你想起上周否掉的一个眼神，对隐迹记忆说：「让怀旧者看看这个。」  
一个鬼影微微变形，像被考古层抬升上来的化石。

---

**傍晚**  
内在合唱团吵起来了。怀疑者嫌「太像某部网大」。  
你没有重写 prompt——你把怀疑者的音量拧大，系统从共振场里**抽走**了过强的「赛博」频率。  
雾变了。不是换模型，是换**调谐**。

---

**夜里**  
你沿时间轴走了一遍情感地形。脚下有两处颠簸。  
AI 建议插入 1.2 秒黑场。你接受。  
然后你做了一件旧工具很难做的事：**大规模结晶**——导出 47 秒 mp4——  
但流体没有干涸。它只是在这一刻**被拍照**。  
项目仍热。明早雾还在。

---

## 四、界面：应该几乎不存在

激进 UI 原则：

1. **默认没有技术视图** — 没有连线、没有端口、没有模型名
2. **手势优先于表单** — 推、拉、捏、停、删
3. **声音与沉默都是输入** — 哼唱、叹气（可选麦克风）、长时间沉默触发「屏息」
4. **界面随创作相位变形**：
   - 探索相 → 雾、调谐盘、地形
   - 收敛相 → 叠加态胶片、合唱团
   - 冻结相 → 时间轴、导出——**此时才像传统剪辑软件**

技术视图（今天的 LocalCanvas 引擎图）是**地下室**，不是客厅。

---

## 五、AI 应该学会什么（能力愿景）

不是「更多模型适配器」，而是这些**内在能力**：

| 能力 | 描述 |
|------|------|
| **共振提取** | 从非语言输入估计跨模态风格向量 |
| **情感物理** | 预测相邻镜头间的情感加速度 |
| **考古检索** | 从否决议题中召回可复活的隐喻 |
| **叠加态管理** | 维护未坍缩版本池，控制算力预算 |
| **共生预测** | 从编辑节奏预测用户意图方向 |
| **负熵编辑** | 识别「假」的元素并建议删减 |
| **结晶编译** | 将流体状态编译为可分发格式（mp4、gif、交互片段） |

## 六、与「现实」的唯一桥梁（可选，不强制）

如果这份愿景要落地到今天的 LocalCanvas 代码库，**只需承认一层翻译关系**：

```
液态创作体验（用户层）
        ↕ 投影 / 编译
现有引擎（节点、DAG、能力 Registry、FFmpeg）
```

- 用户看见的可以是雾与地形
- 地下室仍是节点图——由 AI 自动维护，人可下潜
- **不要求**一步拆掉 React Flow
- **要求**用户默认再也看不见它

这是桥，不是枷锁。若桥妨碍愿景，应拆桥而非削愿景。

## 七、实验里程碑（非工期表）

用**可验证的体验假设**代替「几周完成」：

| 实验 | 假设 | 失败信号 |
|------|------|----------|
| **E1 调谐盘** | 用户更愿意调行星轨道 than 写 200 字 prompt | 7 日内仍打开 prompt 框 |
| **E2 叠加态** | 并行鬼影减少「重做」焦虑 | 重 roll 次数未下降 |
| **E3 情感地形** | 拖拽地形比改分镜表更快收敛 | 用户绕过地形直接生成 |
| **E4 负熵按钮** | 「删除假的」比「再生成」更常用 | 按钮点击率 < 5% |
| **E5 屏息** | 停顿 3 秒后出现幽灵预览提升满意度 | 用户关闭预演 |
| **E6 隐迹** | 召回否决议题产生惊喜感 | 用户手动清空记忆 |
| **E7 结晶** | 导出后回来继续编辑同一「流体」 | 导出后新建项目 |

每个实验可独立立项。**不需要全部成功**才算 v7。  
只要有一个实验彻底改变用户说法——从「我在用 AI 工具」变成「我在创作里」——v7 就成立。

## 八、名字与边界

### 8.1 我们仍坚持的一件事

**意识主权**：你的共振、你的叠加态选择、你的隐迹层、你的合唱团配置——  
默认属于你自己，不属于任何平台的推荐算法。

这与本地或云端无关。  
是**伦理立场**，不是部署架构。

### 8.2 我们主动拥抱的未知

- 交互可能不像「软件」，更像「乐器」或「冥想」
- 创作时长可能从 10 分钟到 10 天——产品不应惩罚长周期项目
- 最终形态可能根本不是「视频编辑器」类目

---

## 九、结语：给未来的读者

> 如果 v7 成功了，没有人会写「LocalCanvas v7 新增了 X 节点」。  
> 他们会说：**「我第一次觉得 AI 在和我一起想，而不是等我下命令。」**

> 如果 v7 失败了，请失败得有趣——  
> 至少我们证明过：创作工具不必永远长得像 1990 年代的流程图。
